HTML Div

Learn how to use the <div> tag to group and structure content in your HTML documents.

Code:

<div style="background-color: #4CAF50; color: white; padding: 20px;">
    <h1>This is a Heading</h1>
    <p>This is a paragraph inside a div element.</p>
</div>

Output:

This is a Heading

This is a paragraph inside a div element.

HTML Colors

Learn how to apply colors using different formats in HTML (e.g., Hex, RGB, Color Names).

Code:

<div style="background-color: #FF5733; color: white;">
    <p>This is a colored div using Hex code.</p>
</div>

Output:

This is a colored div using Hex code.
